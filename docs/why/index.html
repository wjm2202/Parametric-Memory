Technical differentiators

Four architectural choices no other memory system has made.

0.032ms

p95 proof verify

Proof, not trust

Every other memory system asks you to trust that the data hasn't changed. MMPM returns a Merkle proof with every read — verify it yourself in 0.032ms.

64%

cold-start hit rate

Retrieval that predicts

The Markov transition layer learns access patterns and pre-surfaces the next atom before you ask for it — with its own proof attached.

∞

replay depth

Time travel

Every commit increments a monotonic version. Any read accepts asOfVersion or asOfMs — replay the exact memory state at any past moment.

5

atom types

Typed atoms, not blobs

Facts, states, events, relations, procedures — each atom has a semantic type. No embedding required. Structure is the schema.
